快速启动:
  1. ./start.sh

手动启动:
  1. cd backend && npm install && npm run dev
  2. cd frontend && npm install && npm run dev
  3. 访问 http://localhost:5173

生产构建:
  1. cd frontend && npm run build
  2. cd backend && npm start

================================================================================
                        ✨ 项目特色功能
================================================================================

1. 并发数据获取
   使用 Promise.allSettled 并发请求，单个源失败不影响其他源

2. 数据标准化
   统一的数据格式，便于前端展示和扩展新源

3. 响应式设计
   完美适配桌面、平板、手机等各种设备

4. 现代化 UI
   渐变背景、卡片式布局、流畅动画

5. 可扩展架构
   清晰的模块化设计，易于添加新功能和数据源
