# 版本 1.1.0 发布说明

**发布日期**: 2024-11-05  
**版本**: v1.1.0  
**类型**: 功能增强与 UI 优化

---

## 🎉 主要更新

### 1. 🎨 UI/UX 改进

#### 深色主题全面优化
- **问题**: 原有设计使用浅色背景（白色/浅灰）配合深色主题，导致文字对比度不足，部分元素（如标签内的白色文字）不可见
- **解决方案**: 
  - 将所有卡片和 Header 背景改为深色半透明 `rgba(24, 24, 28, 0.95)`
  - 文字颜色统一使用高对比度白色：标题 95%、描述 75%、次要信息 60%
  - Border 和分割线使用 `rgba(255, 255, 255, 0.1)`
  - Hover 效果使用紫蓝色高光 `rgba(99, 102, 241, 0.3)`

#### 视觉效果提升
- 更现代的毛玻璃效果（backdrop-filter）
- 更明显的卡片悬停动画
- 统一的配色方案，提升整体美观度

### 2. 🔍 智能搜索功能

#### 实时关键词搜索
- 在 Header 组件中添加搜索输入框
- 支持搜索新闻标题、描述、作者
- 实时过滤，无需点击搜索按钮
- 大小写不敏感
- 支持清空搜索

#### 技术实现
- 使用 Vue 3 `computed` 属性实现响应式过滤
- 在 App.vue 中集中处理过滤逻辑
- 高性能，不影响页面流畅度

### 3. 🎛️ 数据源选择器

#### 用户自定义数据源
- 用户可自主选择显示哪些数据源的内容
- 支持多选（可同时选择多个数据源）
- 默认全选所有数据源
- 实时过滤，即选即用

#### 支持的数据源
- 🔶 Hacker News - 技术新闻和讨论
- ⭐ GitHub Trending - 热门开源项目
- 📝 Dev.to - 技术博客文章
- 🤖 Reddit - 编程社区讨论
- 💻 CSDN - 中文技术社区（新增）

### 4. 💻 CSDN 中文数据源

#### 新增中文技术社区
- 集成 CSDN 博客排行榜
- 抓取热门技术文章
- 提供中文技术内容，更适合国内开发者
- 包含 fallback 机制，确保服务稳定性

#### 技术实现
- 使用 Cheerio 解析 CSDN 页面
- 多种 CSS 选择器确保兼容性
- 失败时返回占位数据，不影响其他数据源

## 🚀 使用指南

### 关键词搜索
1. 在页面顶部找到搜索框（带有 🔎 图标）
2. 输入关键词（如 "React", "Python", "AI" 等）
3. 页面会实时显示匹配的新闻
4. 点击搜索框右侧的清空按钮可清除搜索

### 数据源筛选
1. 在搜索框下方找到"数据源筛选"区域
2. 勾选或取消勾选数据源复选框
3. 页面会立即显示选中数据源的内容
4. 至少选择一个数据源

### 组合使用
- 可以同时使用搜索和数据源筛选
- 例如：只选择 CSDN，然后搜索 "Vue"
- 结果将是 CSDN 中包含 "Vue" 关键词的文章

## ⚠️ 已知限制

### 当前版本的限制
1. **数据源偏好不持久化**: 刷新页面后恢复默认（全选）
2. **搜索历史未保存**: 无搜索记录功能
3. **CSDN 依赖网站结构**: 若 CSDN 更改页面结构，可能需要更新选择器
4. **无关键词高亮**: 搜索结果中不会高亮显示匹配的关键词

### 计划改进
这些限制将在后续版本中解决，详见 `REQUIREMENTS_CLARIFICATION.md`
